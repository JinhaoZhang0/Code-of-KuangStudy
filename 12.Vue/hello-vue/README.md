# hello-vue

> A Vue.js project

My first Vue.js project combining with ElementUI.

## Dependencies & Plugins

``` bash
# install vue-router
npm install vue-router --save-dev
#install element-ui
npm i element-ui -S
# install dependencies
npm install
# install SASS loader
cnpm install sass-loader node-sass --save-dev
# run to test
npm run dev
```
